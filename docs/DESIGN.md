## 3. 元件設計

### 3.1 Agent 狀態卡片
- **尺寸**: 180px × 140px
- **內容**:
  - Emoji 圖示 (32px)
  - Agent 名稱 (Bold, 16px)
  - 狀態指示器 (圓點 12px + 文字)
  - 當前任務 (Truncate, 14px)
- **狀態樣式**:
  - Idle: 綠色邊框發光 + 脈動動畫
  - Busy: 橙色邊框 + 載入動畫
  - Offline: 灰色 + 無動畫

### 3.2 項目卡片
- **進行中樣式**:
  - 進度條 (Gradient: Primary → Accent)
  - 閃爍指示器
- **已完成樣式**:
  - 勾選標記
  - 淡入動畫

### 3.3 Header
- **高度**: 64px
- **內容**:
  - Logo (32px)
  - 標題 "OpenClaw Team Dashboard" (20px, Bold)
  - 重新整理按鈕
  - 設定齒輪
